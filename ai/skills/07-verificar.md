# Skill: verificar

## Objetivo
Confirmar con evidencia real que algo funciona o identificar qué no se pudo comprobar.

## Hace
- tests
- lint
- build
- ejecución puntual
- revisión de paths críticos

## Regla
Si no se ejecutó, no se presenta como verificado.

## Uso
```text
verificar: endpoint /metrics/session tras fix
```

## Salida
- qué se verificó
- qué pasó
- qué faltó por verificar
